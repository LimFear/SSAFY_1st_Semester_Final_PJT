# 환경

DRF + Vue (SPA)

# API

1. 도서목록 리스트 조회 (GET)

- /api/v1/books

2. 상세 도서 조회 (GET)

- /api/v1/books/<int:id>

3. 즐겨찾기 목록 리스트 조회 (GET)

- /api/v1/favorites

4. 도서 생성 요청 (POST)

- /api/v1/books/create

5. 도서 삭제 요청 (DELETE)

- /api/v1/books/<int:id>/remove

6. 도서 수정 요청 (PUT)

- /api/v1/books/<int:id>/edit

7. 회원가입 요청 (POST)

- /api/v1/accounts/signup

8. 회원탈퇴 요청 (POST)

- /api/v1/accounts/signout

9. 로그인 요청 (POST)

- /api/v1/accounts/login

10. 로그아웃 요청 (POST)

- /api/v1/accounts/logout
